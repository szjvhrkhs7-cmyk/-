# Кошелёк — персональный учёт расходов

Мобильное PWA-приложение для ручного учёта трат и анализа расходов по категориям.

## Возможности

- любое количество пользовательских категорий;
- внесение суммы и даты расхода;
- автоматический итог за выбранный месяц;
- аналитика за месяц, 3 месяца, полугодие и год;
- диаграмма категорий и динамика по месяцам;
- локальное хранение данных на устройстве без сервера;
- экспорт и импорт резервной копии JSON;
- установка на главный экран iPhone как PWA;
- офлайн-режим через service worker.

## Запуск

```bash
npm test
npm run serve
```

После запуска откройте `http://localhost:4173`.

## Установка на iPhone

Откройте опубликованный сайт в Safari, нажмите «Поделиться», затем «На экран Домой». Для установки и офлайн-режима сайт должен быть открыт по HTTPS (исключение — localhost).

## Публикация через GitHub Pages

В настройках репозитория откройте **Settings → Pages**, выберите **Deploy from a branch**, ветку `main` и папку `/ (root)`.

## Хранение и приватность

Записи хранятся только в `localStorage` браузера. Удаление данных Safari или приложения с главного экрана может удалить локальные записи, поэтому периодически используйте экспорт резервной копии.
